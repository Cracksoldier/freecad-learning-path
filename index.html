Everything you need to stay on track

The app is a single HTML file — no install, no account. Open it directly or deploy it on GitHub Pages.

📊
Progress tracking
Two animated progress bars — one for lessons, one for challenges — saved locally in your browser.
📝
Per-lesson notes
Write and save notes inside any lesson. They're stored alongside your progress and shown as a badge on the card.
📤
Export & import
Back up your progress as a JSON file and restore it on any device — no account or cloud required.
🔖
Resume bookmark
Pin any lesson to the top of the page so you can jump straight back where you left off.
🔍
Search
Filter all 27 lessons by keyword instantly. Combine with the level filter to narrow down exactly what you need.
⌨️
Keyboard navigation
Arrow keys navigate between lessons, Escape closes modals, and ? opens the shortcut cheat sheet.
